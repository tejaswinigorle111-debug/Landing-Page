Study Bird Landing Page
 About
This is a single-page landing page created as part of **The Odin Project Foundations** curriculum. The goal of this assignment was to recreate a landing page from a provided design using HTML, CSS, and Flexbox while replacing the placeholder content with meaningful content.

 Technologies Used
- HTML5
- CSS
- Flexbox

 What the Page Includes
- Header with navigation
- Hero section
- Information section
- Quote section
- Call-to-action section
- Footer

 Learning Outcomes
Through this project, I practiced:
- Semantic HTML
- CSS Flexbox
- Page layout and spacing
- Typography and colors
- Building a webpage from a design
